# 🏦 PROJETO – SIMULADOR DE CAIXA ELETRÔNICO 

📅 **Prazo:** (definir)  
📂 **Entrega:** caixa `.py`  
👨‍💻 **Modalidade:** Max 4 integrantes.

----------

## 🎯 Objetivo

Desenvolver um **sistema bancário no terminal (console)** que simule o funcionamento de um caixa eletrônico.

O sistema deve permitir que o usuário realize operações básicas, com **validação de dados e controle de saldo**, aplicando os conceitos estudados em aula.

----------

## 📌 Funcionalidades Obrigatórias

Seu sistema deve conter:

### 🔹 1. Menu Principal (repetitivo)

O programa deve exibir um menu com as opções:

1 – Consultar saldo  
2 – Depositar  
3 – Sacar  
4 – Ver extrato  
5 – Sair

O menu deve continuar aparecendo até que o usuário escolha sair.

----------

### 🔹 2. Saldo Inicial

-   O sistema deve iniciar com um saldo definido no código.
    
-   O saldo deve ser atualizado conforme as operações.
    

----------

### 🔹 3. Depósito

-   Solicitar um valor ao usuário.
    
-   Validar:
    
    -   Entrada numérica.
        
    -   Valor positivo.
        
-   Atualizar o saldo.
    
-   Registrar a operação no extrato.
    

----------

### 🔹 4. Saque

-   Solicitar um valor ao usuário.
    
-   Validar:
    
    -   Entrada numérica.
        
    -   Valor positivo.
        
    -   Saldo suficiente.
        
-   Atualizar o saldo.
    
-   Registrar a operação no extrato.
    

----------

### 🔹 5. Extrato

-   Armazenar todas as movimentações em uma **lista**.
    
-   Mostrar todas as operações realizadas.
    
-   Caso não existam operações, informar isso ao usuário.
    

----------

## ⚠️ Regras Técnicas Obrigatórias

Seu código deve conter obrigatoriamente:

✔ Uso de funções (mínimo 3)  
✔ Estruturas condicionais  
✔ Estruturas de repetição  
✔ Lista para armazenar extrato  
✔ Validação de erros de entrada  
✔ Código organizado e indentado corretamente

----------

## 🚫 Não é permitido

❌ Interface gráfica  
❌ Uso de banco de dados  
❌ Bibliotecas externas  
❌ Colocar todo o código dentro de uma única função  
❌ Copiar código pronto
